이것을 가정 하에 가중치가 동일한 1:n 그래프의 경로, 촌 을 포함한 bfs 로 구현, 값 반환 을 계획 했다
하지만 20%, 95% 에서 틀렸으며 아마 테스트케이스 에서는 다양한 규칙의 노드가 나온것 같다
